# Avocado

This repo contains the contracts for Avocado Exchange, a good faith fork of Velodrome Finance.

## Testing

This repo uses both Foundry (for Solidity testing) and Hardhat (for deployment).

Foundry Setup

```ml
forge init
forge build
forge test
```

Hardhat Setup

```ml
npm i
npx hardhat compile
```
